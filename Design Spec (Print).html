Avatar / Icon / Input

  • Avatar: 정원형. size 24~60, initials 또는 src, ring 옵션 (보라 1.5px outline)
  • Icon: 단일 SVG 정의. 24×24 viewBox, strokeWidth default 1.75. 사용 가능한 name: home, calendar, chat, compass, user, search, bell, cart, heart, chevronRight, chevronLeft, chevronDown, close, plus, check, star, map, clock, location, play, wave, settings, arrowUp, eye
  • Input: label / placeholder / suffix. focus 시 --sh-focus ring

BottomTab (하단 내비)

5탭 고정 — 홈 · 예약 · 커뮤니티 · 투어 · 마이버클. 높이 72px (+ paddingBottom 14 for safe area). Active 색 --c-brand, inactive --c-text-4. Backdrop-blur 적용.

TabHeader (페이지 내 탭)

언더라인 스타일. 인디케이터는 ink (#101114), 보라 X. 활성 텍스트 ink, 비활성 text-3.

🆕 신규 — Dive Points (커뮤니티 다이빙 포인트)

기존 없는 화면. 신규 라우트 /community/dive-points 권장.

  • 피드 — 스토리 레일 + 3가지 포스트 패턴 (단일사진 IG / 텍스트 Threads / 그리드)
  • 포인트 페이지 — IG 프로필 스타일 + grid/feed/log/tag 4탭
  • 다이브 로그 작성 — Threads 컴포저 + 수심/시야/수온/생물 입력

🆕 신규 — 첫 사용자 튜토리얼

로그인 직후 첫 홈 진입 시 1회. localStorage.buccl_tutorial_seen_v1 플래그로 관리. Deep Sea 다크 톤 (시그니처 보라 X).

  1. Welcome — 풀블리드 다크 인트로, 헤드라인 + 시안 점 피처 리스트 + "둘러보기" CTA
  2. 코치마크 1: 강습 — 강습 예약 아이콘. ↳ "수강권에서 강습권도 함께 구매할 수 있어요"
  3. 코치마크 2: 연습반 — ↳ "내 자격증 메뉴에서 이미 보유한 자격증도 등록"
  4. 코치마크 3: 투어
  5. 코치마크 4: 커뮤니티 — 다이브 로그 / 후기

컴포넌트: SpotlightV2 (radial-gradient 컷아웃 + 흰 펄스 링 2겹) / TopProgress / BottomDock / TooltipV2 (다크 #101820 + 시안 칩)

DELIVERY

구현 권장 순서 & 체크리스트

Phase 1 — Foundation (1주)

  1. 📦 tokens.css → _tokens.scss 변환 · FE/src/assets/scss/_tokens.scss 신규
  2. 📦 공용 컴포넌트 리뉴얼 · Button, Badge, Avatar, Icon, BottomTab, SubHeader, TabHeader
  3. 🔍 기존 보라/네이비 혼용 색상 검수 · 모두 --c-brand 로 통일

Phase 2 — Main flow (2주)

  1. 🏠 홈 (MainPage) · QuickMenu 10개 PNG + BrandPicker + 인기 커뮤니티 + 푸터
  2. 📅 예약 / 투어 캘린더 · 컴팩트 캘린더 + 수강권 컴팩트 배너
  3. 💬 커뮤니티 · 패딩 16px 통일 + 8px gap

Phase 3 — New features (3주)

  1. 🌊 Dive Points · 신규 라우트, 3개 화면 (피드/포인트/컴포저)
  2. 👋 첫 사용자 튜토리얼 · components/onboarding/HomeTutorial.jsx 신규, localStorage 플래그

Phase 4 — 나머지 화면 (2주)

  1. 📋 상세 페이지 (강습/투어/상품/커뮤니티 글)
  2. 🛒 커머스 플로우 (장바구니 → 결제 → 주문)
  3. 👤 계정 (로그인/가입/찾기/마이) + 알림/설정

접근성 & 품질 체크

  • 모든 인터랙티브 요소 hit target 최소 44×44px
  • 텍스트 대비 WCAG AA (text-1 / text-2 모두 통과 확인됨)
  • 포커스 링 --sh-focus 일관 적용
  • 다크모드는 이번 라운드 범위 외 — 단, Welcome 화면은 다크 톤이므로 단독 분리

질문 & 컨택

마이크로 인터랙션 디테일, 빈 상태 일러스트, 실제 사진 자산이 필요한 경우 디자이너에게 직접 문의 부탁드립니다.
